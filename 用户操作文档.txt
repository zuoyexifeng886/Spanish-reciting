西语背单词 - 用户操作文档
========================

【产品简介】
一款纯离线的西班牙语背单词网页工具，所有数据保存在浏览器本地，无需联网即可使用。
支持艾宾浩斯遗忘曲线复习、多种自测模式、错题本等功能。

========================
一、快速开始
========================

1. 打开 index.html 即可使用，无需安装
2. 首次使用建议先加载示例词库体验功能
3. 支持手机端和电脑端，建议使用 Chrome / Edge / Safari 浏览器

========================
二、词库管理
========================

【入口】底部导航 → 词库

2.1 加载示例词库
- 点击「加载示例词库」按钮
- 内置 28 个示例单词，覆盖日常、旅游、商务、DELE四大分类
- 适合首次使用快速体验

2.2 手动添加单词
- 点击右上角「+」按钮
- 填写字段：
  * 西语单词（必填）
  * 词性（可选）
  * 中文释义（必填）
  * 动词变位（可选，动词填写）
  * 场景例句（可选）
  * 难度等级（A1/A2/B1/B2/C1）
  * 分类（日常基础/旅游/商务/DELE真题）
  * 高频词标记（可选）
- 点击「保存」完成添加

2.3 编辑单词
- 在单词列表中点击「✏️」按钮
- 修改后点击「保存」

2.4 删除单词
- 在单词列表中点击「🗑️」按钮
- 确认后删除

2.5 导入Excel词库
- 准备CSV格式的词库文件（Excel可另存为CSV）
- 列顺序：单词, 词性, 中文释义, 动词变位, 例句, 难度等级, 分类
- 第一行可以是表头，会自动跳过
- 点击「导入词库」选择文件即可

2.6 导出词库备份
- 点击「导出备份」按钮
- 自动下载 CSV 格式文件
- 可用 Excel 打开编辑

2.7 搜索和筛选
- 顶部搜索框：支持西语/中文搜索
- 分类标签：按分类筛选
- 难度下拉：按A1-C1等级筛选

========================
三、背诵学习
========================

【入口】底部导航 → 学习

3.1 学习流程
- 系统自动安排今日待复习单词 + 今日新词
- 先复习旧词，再学习新词
- 点击卡片翻转查看释义
- 点击「🔊」按钮听发音

3.2 记忆评级
学习完每个单词后，根据记忆程度选择评级：
- 😰 没记住 → 1天后复习
- 🤔 有点印象 → 3天后复习
- 😊 记住了 → 7天后复习
- 🤩 非常熟 → 15天后复习

系统根据艾宾浩斯遗忘曲线自动安排下次复习时间。

3.3 学习完成
- 全部单词学完后显示完成页面
- 可查看今日学习统计
- 可返回首页或去做练习

========================
四、自测练习
========================

【入口】底部导航 → 练习

4.1 四种练习模式
1. 看西语写中文
   - 给出西语单词，默写中文释义
   - 支持模糊匹配

2. 看中文写西语
   - 给出中文释义，默写西语单词
   - 自动忽略重音和大小写差异

3. 听力听写
   - 播放西语发音，听写单词
   - 可反复点击播放
   - 自动忽略重音和大小写差异

4. 选择题
   - 给出西语单词，4选1中文释义
   - 即时反馈对错

4.2 练习设置
- 题目数量：10/20/30/50题
- 词库范围：全部单词/已学单词/错题本

4.3 练习结果
- 显示正确率、正确数、错误数
- 答错的单词自动加入错题本
- 可重新练习或返回首页

========================
五、错题本
========================

【入口】首页 → 错题本 或 练习页错题专项

5.1 功能说明
- 练习中答错的单词自动收录
- 记录每个单词的错误次数
- 支持按「全部/今日/高频」筛选

5.2 错题专项练习
- 点击「错题专项练习」按钮
- 只从错题本中抽题练习
- 针对性巩固薄弱单词

5.3 清空错题本
- 点击「清空错题本」
- 确认后清空所有错题记录

========================
六、设置与数据管理
========================

【入口】底部导航 → 我的

6.1 学习设置
- 每日新词数量：5-50个可调
- 发音语速：慢速/正常/快速
- 自动播放发音：开/关
- 显示例句：开/关
- 显示动词变位：开/关

6.2 数据管理
- 导出全部数据：导出JSON格式完整备份（含词库+学习记录+设置）
- 导入数据备份：从JSON文件恢复所有数据
- 重置学习进度：清空学习记录，保留词库
- 清空所有数据：删除所有数据（不可恢复）

========================
七、常见问题
========================

Q1: 数据会丢失吗？
A: 所有数据保存在浏览器本地存储中。清除浏览器数据或更换浏览器会丢失，建议定期导出备份。

Q2: 发音没声音怎么办？
A: 确保浏览器支持语音合成（Chrome/Edge/Safari都支持），检查系统音量。部分浏览器需要先有用户交互才能播放声音。

Q3: 支持哪些浏览器？
A: 推荐 Chrome、Edge、Safari 最新版本。不支持 IE 浏览器。

Q4: 手机上怎么用？
A: 直接用手机浏览器打开 index.html 即可，界面已适配移动端。也可以添加到主屏幕像APP一样使用。

Q5: 怎么导入自己的词库？
A: 用Excel整理好单词，另存为CSV格式（UTF-8编码），然后在词库页点击导入。

Q6: 艾宾浩斯复习是怎么算的？
A: 根据你选择的记忆评级，分别在1/3/7/15天后安排复习。随着掌握程度提高，复习间隔会越来越长。
